PUC Minas - Bakta

Interface web para anotação genômica bacteriana utilizando a ferramenta Bakta.

Tecnologias
  • Backend: Flask + Python
  • Frontend: Bootstrap 5 + JavaScript
  • Pipeline: Bakta + BioPython
  • Container: Docker
